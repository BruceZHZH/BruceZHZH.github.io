windows平台下利用Github和Hexo搭建个人博客

10月 02, 2016

背景

Hexo 是一个简单地、轻量地、基于Node的一个静态博客框架，可以方便的生成静态网页托管在github上。是基于Node.js的一个简单且功能强大的博客框架。

GitHub Pages 可以被认为是用户编写的、托管在github上的静态网页。它的空间免费且稳定（不需要翻墙）， 可以用于介绍托管在github上的项目或者搭建网站。

开始部署

需要用到的工具

  • Node.js
  • Git/Github
  • Sublime

1. 安装Git

2. 安装Node.js

3. 安装Hexo

打开Git Bash，进入一个目录，使用npm指令安装Hexo

Hexo安装完成之后初始化hexo

安装依赖的包

到此Hexo已经安装完成，可以在本地启动服务,访问==localhost:4000==查看站点的效果

4. 部署到GitHUb上

在Github上新建repository，在创建的时候需要注意仓库的名称需要是

==Github账户名.github.io==

仓库建立好之后选择仓库的设置,选择Launch automatic page generator

5. 修改Hexo的配置文件

之前初始化的Hexo文件件下_config.yml文件里配置着站点的一些信息，将最后的deploy项做一些修改

修改好后进入gitbash

待Git提交完所有内容后，在浏览去上访问GitHub Pages就可以看到Hexo的内容。

Hexo还有很多好看的主题可以更换，Hexo主题-知乎

个人使用的是一个叫tranquilpeak的主题。Hexo也支持很多插件，比如多说评论、分享、谷歌数据统计等。
